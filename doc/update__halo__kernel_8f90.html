Fortran kernel to update the external halo cells at the bottom of a patch.

Author:
Wayne Gaudin, David Beckingsale

Updates halo cells for the required fields at the required depth for any halo cells that lie on an external boundary. The location and type of data governs how this is carried out. External boundaries are always reflective.
