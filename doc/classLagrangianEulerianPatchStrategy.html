Detailed Description

A class to describe the abstract methods needed by LagrangianEulerianLevelIntegrator.

The abstract methods here provide all the operations needed to integrate a patch through the Lagrangian-Eulerian scheme seen in Cloverleaf. This includes the Lagrangian-step methods to advance the solution, and the advection methods for remapping the grid.
