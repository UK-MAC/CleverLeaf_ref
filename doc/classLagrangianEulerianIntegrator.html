Detailed Description

Controls the steps necessary to advance the solution across the entire adaptive mesh.

Input Parameters

Definitions:

  • start_time the start time of the simulation.
  • end_time the end time of the simulation.
  • end_step the end step of the simulation.
  • initial_dt the initial dt value.
  • max_dt the maximum dt value.
  • grow_dt the maximum factor to increase the dt by each time step.
  • fix_dt whether or not to use a fixed dt. If true, the max_dt value is used for every timestep.
  • regrid_interval how often to regrid the mesh.

Details:

parameter type default
start_time double 0.0
end_time double 1.0
end_step int 100000000
initial_dt double 0.04
max_dt double tbox::MathUtilities<double>::getMax()
grow_dt double 1.5
fix_dt bool FALSE
regrid_interval int 1

A sample input file entry might look like this:

   start_time = 0.e0
   end_time = 10e0
   grow_dt = 1.5e0
   max_integrator_steps=87
   regrid_interval=5
